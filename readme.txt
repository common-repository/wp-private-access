Wp Private Access adds an http authentication dialog using WordPress users database

== Description ==

Wp Private Access adds an http authentication dialog using WordPress users database. Wp private is particularly useful for testing blogs
and websites based on WordPress keeping them visibile only to authorized users.

The plugin provides also with a template page for unauthorized access (401 error). 

== Installation ==

Upload the Wp Private Access plugin to your blog, Activate it. Enjoy ;)
